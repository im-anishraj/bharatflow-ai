# Support

## Questions

Use [GitHub Discussions](https://github.com/im-anishraj/bharatflow-ai/discussions) for setup questions, contribution ideas, architecture questions, and roadmap discussion.

## Bugs

Use the bug report issue template and include:

- app or package affected;
- branch or commit;
- reproduction steps;
- expected and actual behavior;
- screenshots, logs, or a short screen recording when useful.

## Security

Do not open public issues for vulnerabilities. Follow [SECURITY.md](SECURITY.md).

## Commercial Or Enterprise Use

BharatFlow AI is not yet cleared for commercial hosted use from this repository. For enterprise pilot questions or legal/commercial-path discussion, contact [Anish Raj](mailto:anishraj.dev@gmail.com).
