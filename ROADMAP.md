# Roadmap

This roadmap is intentionally public so contributors can find useful work without guessing.

## MVP

- Keep the Typebot-derived monorepo stable.
- Polish BharatFlow AI landing, pricing, templates, and legal surfaces.
- Add India-focused templates for WhatsApp lead qualification, site visits, admissions, KYC intake, support triage, payment links, and CSAT.
- Improve contributor documentation and reproducible local setup.
- Maintain CI, CodeQL, Dependabot, branch protection, and issue triage discipline.

## v1

- Add India-native connectors as Forge blocks where possible.
- Improve DPDP-aware workflow patterns for consent, purpose, retention, and handoff.
- Add better test coverage around templates, viewer runtime smoke paths, and landing content.
- Document deployment patterns for self-hosted pilots.
- Harden observability, backup, and incident-response runbooks.

## Scale-Up

- Evaluate a legally cleared commercial path.
- Add enterprise audit, retention, and admin controls if the product direction is approved.
- Build a maintained connector ecosystem for CRM, payments, telephony, support, and document workflows in India.
- Publish production-quality docs, examples, and demo data.
